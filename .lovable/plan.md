

# Timeless Moments Photo Studio — One-Page Website

## Design & Theme
- **Color palette**: Blush pinks, creamy whites, soft golds, subtle lavenders — warm romantic pastels
- **Typography**: Playfair Display (serif) for headings, Montserrat (sans-serif) for body text via Google Fonts
- **Animations**: Fade-in on scroll, parallax effects, heart-shaped hover animations on cards, smooth section transitions
- **Overall feel**: Intimate, elegant, like a digital wedding invitation

## Sections (single scrolling page)

### 1. Sticky Navbar
- Transparent on top, solid on scroll with blur backdrop
- Links: Home, About, Packages, Gallery, Contact
- Mobile hamburger menu with slide-in drawer

### 2. Hero Section
- Full-screen slideshow of romantic wedding placeholder images with dreamy overlay
- Centered text: studio name, tagline "Capture Love That Lasts Forever", romantic subtitle
- Animated "Book Now" CTA that smooth-scrolls to Contact
- Subtle floating heart particles animation

### 3. About Us Section
- Warm intro text about the studio, based in Chennai
- 2-3 placeholder team/studio photos with soft rounded frames
- 3 icon cards: "Romantic Storytelling", "Timeless Quality", "Personalized Experience"
- "Our Story" button

### 4. Packages Section
- 4 pricing cards in responsive grid: Sweetheart (₹25,000), Forever (₹45,000), Eternal (₹75,000), Custom Love
- Cards lift on hover with glowing heart accent animation
- Each card lists features and has a "Book Now" button
- "Forever" package highlighted as popular

### 5. Gallery Section
- Category tabs: Weddings, Engagements, Portraits
- Masonry-style grid of 12+ placeholder images with dreamy filters
- Lightbox modal on click with smooth scale-in transition and navigation

### 6. Contact Section
- Form with fields: Name, Email, Phone, Message, Date Preference
- WhatsApp and Email CTA buttons
- Embedded Google Maps placeholder for Chennai location
- Contact details: phone, email, Instagram link
- "Let's create magic together!" heading with heart icon

### 7. Footer
- Social media icons (Instagram, Facebook, Pinterest)
- Copyright notice
- "Made with ❤️" text

## Technical Approach
- All sections in a single Index page with smooth scroll behavior
- Intersection Observer for scroll-triggered fade-in animations
- CSS keyframes for heart hover effects and floating particles
- Responsive mobile-first layout using Tailwind
- SEO meta tags in index.html
- Placeholder images from Unsplash for wedding/portrait photography
